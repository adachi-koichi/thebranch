# 組織ダッシュボード設計 (THEBRANCH)

## 概要

**利用者**: 組織管理者（Organization Admin）  
**目的**: 組織内のエージェント・プロジェクト・リソースの統合監視と管理  
**対象**: THEBRANCH プロダクト v1.0 以降

---

## 1. 主要パネル構成

### 1.1 ヘッダー
- **組織名・ロゴ表示**
  - 現在ログイン中の組織名
  - 環境表示（本番 / ステージング / 開発）
- **ユーザーメニュー** 
  - プロフィール
  - 設定
  - ログアウト

### 1.2 左サイドバー（ナビゲーション）

```
📊 ダッシュボード（デフォルト）
├─ 📈 統計・メトリクス
├─ 👥 メンバー管理
├─ 🤖 エージェント管理
├─ 📁 プロジェクト管理
├─ 🔐 セキュリティ・監査
├─ ⚙️ 組織設定
└─ 📋 インベントリ
```

## 2. メインパネル（ダッシュボード）

### 2.1 ステータスサマリー（上部 4列）

```
┌──────────────┬──────────────┬──────────────┬──────────────┐
│ アクティブ   │ プロジェクト  │ メンバー     │ リソース使用 │
│ エージェント  │ (進行中)     │ (アクティブ) │ (CPU/Mem)   │
│    12 / 15   │     8 / 20   │    45 / 100  │  65% / 42%   │
│ ↑ +3 (24h)  │ ↑ +2 (24h)  │ ↓ -1 (24h)  │  → +5% (1h) │
└──────────────┴──────────────┴──────────────┴──────────────┘
```

**表示内容**:
- アクティブエージェント数（合計に対する比率）
- 進行中のプロジェクト数（合計に対する比率）
- アクティブメンバー数（ログイン状態）
- リソース使用率（CPU / メモリ / ストレージ）
- 24時間・1時間の増減トレンド

### 2.2 エージェント監視パネル（左下）

```
┌─────────────────────────────────────┐
│ アクティブエージェント              │
├─────────────────────────────────────┤
│ Name          │ Role     │ Status    │
├─────────────────────────────────────┤
│ orchestrator  │ Orch.    │ 🟢 稼働中 │
│ pm@exp-stock  │ PM       │ 🟢 稼働中 │
│ em@breast-ca… │ EM       │ 🟡 待機中 │
│ eng@ai-orch…  │ Eng.     │ 🔴 停止  │
│ ...           │ ...      │ ...      │
└─────────────────────────────────────┘
```

**列**:
- **Name**: セッション名 / エージェント識別子
- **Role**: Orchestrator / PM / EM / Engineer / QA
- **Status**: 🟢 稼働中 / 🟡 待機中 / 🔴 停止 / ⚠️ エラー
- **クリック**: 詳細パネルを開く

### 2.3 プロジェクト進捗パネル（右上）

```
┌──────────────────────────────────────┐
│ 進行中のプロジェクト                  │
├──────────────────────────────────────┤
```

**表示内容**:
- プロジェクト名
- 進捗バー（% / 期間）
- タスク完了数 / 合計数
- 遅延フラグ（赤色）

## 3. 機能・インタラクション

### 3.1 フィルタリング
- **エージェント状態**: 稼働中 / 待機中 / 停止 / エラー（複数選択可）
- **プロジェクト状態**: 計画中 / 進行中 / 完了 / 保留中
- **時間範囲**: 24h / 7d / 30d / カスタム

### 3.2 検索
- **グローバル検索バー**: プロジェクト / メンバー / エージェント名で検索
- **タグ検索**: 組織、チーム、優先度でフィルタ

### 3.3 レスポンシブ設計
- **デスクトップ** (1920px+): 4パネル表示（サマリー / エージェント / プロジェクト / アラート）
- **タブレット** (1024px-1920px): 2×2 グリッド
- **モバイル** (<1024px): スタック表示（縦方向）

### 3.4 自動更新
- **ポーリング間隔**: 30秒（ステータス）/ 5分（メトリクス）
- **WebSocket**: リアルタイムアラート（オプション）

| ロール | 表示 | 編集 | 削除 |
|---|---|---|---|
| Organization Admin | ✅ 全て | ✅ 全て | ✅ 全て |
| Project Manager | ✅ 自プロジェクト | ✅ 自プロジェクト | ✅ 自プロジェクト |
