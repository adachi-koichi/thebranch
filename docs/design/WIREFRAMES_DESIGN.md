```
[━━━━━━━━━━━━━━━━━━━]  (border: red, icon: ❌)
❌ このメールアドレスは既に登録されています

成功時:
[━━━━━━━━━━━━━━━━━━━]  (border: green, icon: ✓)
```

### 3.3 ホバー・クリック効果

**カード** (タスク・部署):
- ホバー: shadow 追加, scale: 1.02
- クリック: opacity: 0.9 → 詳細ページ遷移

**ボタン**:
- ホバー: background-color 濃くなる
- クリック: active state 表示
- トランジション: 200ms ease-out

**テーブル行**:
- ホバー: background-color: gray-50
- クリック: 詳細ページ遷移
