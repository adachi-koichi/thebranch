# THEBRANCH ユーザージャーニーマップ

**作成日**: 2026-04-23  
**バージョン**: 1.0  
**対象ペルソナ**: P1（起業家）/ P2（複業マネジャー）/ P3（営業責任者）

---

## 概要

このドキュメントは、THEBRANCHの3つの主要ペルソナがどのようなジャーニーを経て、プロダクトビジョン「ひとりがユニコーンになれる時代へ」を実現するかを段階的に記述したものです。

各ペルソナのジャーニーは以下の5段階で構成されます：

1. **Awareness（認識）** - THEBRANCHの存在を知る
2. **Onboarding（導入）** - セットアップ・初期設定
3. **Activation（実行）** - 実際に部署・チーム運営を開始
4. **Retention（継続）** - 日常的な運用・改善
5. **Advocacy（推奨）** - 他者への推奨・スケーリング

**AI Q&A 実行例**

ユーザーが質問: 「高額商材の営業戦略は?」

```
Q: 高額商材の営業戦略は?

AI A: 
【戦略分析】
  1. リード特性分析
     ├─ 企業規模: 50名以上（高額化可能）
     ├─ 業種: SaaS / コンサル（導入意欲高）
     └─ 意思決定者: CFO / VPOps（確度重視）

  2. 推奨プロセス
     ├─ 段階1（1週）: ニーズヒアリング / ROI試算
     ├─ 段階2（2週）: デモ / 事例紹介
     └─ 段階3（3-4週）: 提案 / 契約

  3. あなたのリード分析
     ├─ 高額化可能: 5件（うち3件が適切段階）
     ├─ 推奨: 段階1 向け初期接触を即座に実行
     └─ [実行テンプレート] [詳細分析] ボタン

所要時間: 3秒（リアルタイム AI 分析）
```

**成功基準**
- ✅ ダッシュボード表示時間 < 2秒
- ✅ AI取得リード数 ≥ 10件
- ✅ AI Q&A 応答時間 < 5秒
- ✅ ユーザーがテンプレート実行 ≥ 1回

### ジャーニーステージ4: Retention（月4日～30日）

**シナリオ**: 日常的に営業プロセスを回し、AI Agent からの推奨を活用

| 要素 | 詳細 |
|---|---|
| **タッチポイント** | 日次ダッシュボード / 週次レポート / KPI 分析 |
| **行動** | 毎朝ダッシュボード確認 → AI推奨アクション実行 → 成果記録 → 週次レビュー |
| **感情** | 😊 満足 → 🚀 成長実感 |
| **ペイン** | 手動作業の削減 / AI Agent の精度向上 |
| **期待** | 営業プロセスの自動化 / スケーリング |
| **ゴール** | 月目標達成 / 次月への組織スケーリング |

**週次実行フロー**

```
【週1サイクル】

月曜朝
  ├─ 先週の成果確認: リード15件 → 成約3件（20%）
  ├─ KPI分析: 「段階2（デモ）での成約率が高い」と AI分析
  └─ 推奨: 「段階2への進捗を加速」

火～木
  ├─ 日次ダッシュボード確認（朝5分）
  ├─ AI推奨アクション実行（午前1時間）
  ├─ Kanban Board更新（常時）
  └─ AI Q&Aで疑問解決（オンデマンド）

金曜午後
  ├─ 週次レポート確認
  │  ├─ 成約数: 4件（月間目標 20件中 40% 達成）
  │  ├─ 平均営業期間: 10日（目標 14日より短い ✅）
  │  ├─ 営業費用: ¥20k（月間予算 ¥100k中）
  │  └─ ROI: 初月でも黒字予測
  │
  ├─ AI推奨: 「リード数を増やして、スケーリングを加速できます」
  │
  └─ アクション: 「営業メンバーを追加採用」の検討開始

日曜夜
  ├─ 全メンバーへ週次サマリーメール送信（自動）
  └─ 月曜朝の改善方針を AI が提案
```

**チーム拡張への流れ**

```
【4週目：スケーリング判断】

ユーザー（起業家）の思考
  「1人での営業で月目標の50%達成。あと2人追加すれば月目標達成できる!」

AI推奨
  「現在の生産性から、営業2名追加で月目標達成可能。
   新メンバー用にオンボーディングテンプレートを自動生成できます」

アクション
  ├─ [営業テンプレート複製]
  │  └─ 新メンバー2人分の初期設定自動生成
  │
  └─ [メンバー追加オンボーディング]
     ├─ SalesBot の指示をメンバー用にカスタマイズ
     ├─ 既存のナレッジ（営業戦略・トークスクリプト）を共有
     └─ メンバー2人が「同じレベル」で営業開始

結果: 4人営業チームが 5日で準備完了 ✅
```

**成功基準**
- ✅ 日次ダッシュボード利用率 > 80%
- ✅ AI推奨アクション実行率 > 60%
- ✅ KPI達成率 ≥ 80%（月目標）
- ✅ チーム拡張への検討 = Yes

### ジャーニーステージ5: Advocacy（月2～3ヶ月目）

**シナリオ**: 営業部の成功を見て、他部署（CS、企画）への拡張を検討 → 他の起業家に推奨

| **感情** | 🌟 成功実感 → 🎯 拡張への確信 |
| **ペイン** | 部署ごとの最適化 / ナレッジ管理 |
| **期待** | 全社規模への拡張 / 組織構築の加速 |
| **ゴール** | 複数部署の運営 / 業界への認知向上 |

**実行フロー**

```
【2ヶ月目】営業組織の成功確定

KPI確認
  ├─ 月売上: ¥1,500k（目標 ¥1,000k を達成 ✅）
  ├─ 営業チーム規模: 4人（うち新規2人は初月から貢献）
  ├─ 営業周期: 平均 9日（業界平均 30日より大幅短縮）
  └─ 営業人件費: ¥300k/月（利益 ¥1,200k）

決定: 「CS部署も立ち上げて、顧客ロックイン率を上げたい」

アクション
  ├─ THEBRANCH上で [CS テンプレート] を検索・適用
  ├─ 営業チームから CS へ人材転配 1人
  ├─ AI Agent (CSBot) を自動配置
  └─ 3日で CS部署が稼働開始 ✅

結果: 2部門運営 / 営業+CS = 総6人の組織体制確立
```

**外部への推奨活動**

```

アクティビティ
  ├─ LinkedIn: 「AI営業組織で売上1.5倍に」とポスト
```
