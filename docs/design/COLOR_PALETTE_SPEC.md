## 1. プライマリカラー（メイン色）

### 概要
THEBRANCHのブランド色。主要なCTA、リンク、アクティブ状態に使用します。

### カラーシステム

| レベル | 色値 | 用途 | 備考 |
|---|---|---|---|
| **primary-50** | `#EFF6FF` | 背景（ライト） | ホバー背景、弱調強調 |
| **primary-100** | `#DBEAFE` | 背景（ライト）のバリアント | テーブルヘッダー背景 |
| **primary-200** | `#BFDBFE` | ボーダー / ライト背景 | ディスabled要素の背景 |
| **primary-300** | `#93C5FD` | ライトボーダー | フォーカス外の入力ボーダー |
| **primary-400** | `#60A5FA` | ライト強調 | プログレスバー（中程度） |
| **primary-500** | `#3B82F6` | **メインカラー** | CTA / リンク / アイコン |
| **primary-600** | `#2563EB` | ホバー状態 | ボタンホバー効果 |
| **primary-700** | `#1D4ED8` | アクティブ状態 | ボタンクリック時 |
| **primary-800** | `#1E40AF` | ダーク強調 | テキストリンク（訪問済み） |
| **primary-900** | `#1E3A8A` | 最も濃い | 将来予約 / 拡張用 |

### 使用例

```css
/* CTA ボタン */
.btn-primary {
  background-color: #3B82F6; /* primary-500 */
  color: #ffffff;
}

.btn-primary:hover {
  background-color: #2563EB; /* primary-600 */
}

.btn-primary:active {
  background-color: #1D4ED8; /* primary-700 */
}

/* フォーカス状態 */
input:focus {
  border-color: #3B82F6; /* primary-500 */
  box-shadow: 0 0 0 3px rgba(59, 130, 246, 0.1); /* primary-500 透明度10% */
}
```

## 3. ニュートラルカラー（グレースケール）

| レベル | 色値 | 用途 |
|---|---|---|
| **gray-50** | `#FAFAFA` | ページ背景 |
| **gray-100** | `#F3F4F6` | 淡い背景（ライト） |
| **gray-200** | `#E5E7EB` | ボーダー・区切り線 |
| **gray-300** | `#D1D5DB` | ボーダー・区切り線（濃） |
| **gray-400** | `#9CA3AF` | セカンダリテキスト（薄） |

| **gray-700** | `#374151` | ボディテキスト（標準） |

### 使用例

```css
/* ボディテキスト */

/* ラベル・キャプション */

/* 無効なテキスト */
.disabled { color: #9CA3AF; /* gray-400 */ }

/* ボーダー */
.border { border-color: #E5E7EB; /* gray-200 */ }
```

## 6. アクセシビリティ基準

### コントラスト比

すべてのカラー組み合わせは **WCAG 2.1 AA 基準** に準拠:

| 組み合わせ | コントラスト比 | 準拠レベル |
|---|---|---|
| primary-500 on white | 4.5:1 | ✓ AA |
| gray-600 on white | 7.5:1 | ✓ AAA |
| error-500 on white | 5.5:1 | ✓ AA |
| warning-500 on white | 7.0:1 | ✓ AAA |

### カラーのみに依存しない設計

- ✓ ステータスはアイコン + 色の組み合わせ
- ✓ エラーメッセージはテキスト + 色
- ✓ チャート凡例にパターンを含める
